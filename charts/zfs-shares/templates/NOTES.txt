NOTE ON UPGRADES: the ZfsShare CRD is installed from the chart's crds/ directory.
`helm upgrade` does NOT update it (no Helm flag does). After upgrading to a
release that changes the CRD schema, apply the CRD manually first:

     kubectl apply -f charts/zfs-shares/crds/     # local checkout; or `make install-crd`
     # published chart: helm pull ... --untar && kubectl apply -f zfs-shares/crds/
